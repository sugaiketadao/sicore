Class Information

Class Name: StorageUtil

Description

Session storage utility class.
  • Store and retrieve associative arrays in browser session storage by the following scope + key:
    • Page scope (per URL HTML file, data persists within one page)
    • Module scope (per URL module directory, data shared across pages)
    • System scope (data shared across entire system)
  • Assumes non-critical processing, so generally does not throw exception errors.
