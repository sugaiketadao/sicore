Class Information

Class Name: StorageUtil

Description

Session storage utility class.
  • Stores and retrieves associative arrays in browser's session storage with the following units + keys.
    • Page unit (HTML file unit of URL, data retention within one page)
    • Module unit (module directory unit of URL, data sharing between pages)
    • System unit (data sharing across entire system)
  • Assumes non-critical processing and does not throw exceptions in principle.
